# Simplest Ridealong

This is the simplest possible ridealong demonstration. It contains just a few basic commands.

## What is a Ridealong?

A ridealong is a markdown file with annotated code blocks that can be automatically loaded and executed in federation-command. The user steps through each command one at a time.

## Example Commands

The following commands will be loaded when you run `ridealong simplest.md`:

```ridealong
echo "Hello from ridealong!"
echo "This is the second command"
ls -la
echo "Ridealong complete!"
```

Press Enter to execute each command, or use Up/Down to select "exit" from the menu.
